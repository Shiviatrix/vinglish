01 / FOUNDATIONS

Getting started

Source files use .ving. Check, run, build, and format with the vng driver.

vng check hello.ving
vng run hello.ving
vng build hello.ving --output hello
use std.io
function main()
begin
  println("Hello from Vinglish.")
end

02 / DATA

Types and records

Use type for records. Fields are named, checked, and lowered to deterministic layouts.

type Point
begin
  x: number
  y: number
end

The scalar vocabulary is number, decimal, text, boolean, and address<T>. Use borrow and borrow mutable to state aliasing intent.

03 / CONTROL

Control flow reads directly.

repeat while current <= limit
begin
  total be total + current
  current be current + 1
end

if, repeat while, and match lower to explicit control-flow blocks before optimization.

04 / REUSE

Generics stay explicit.

Use concrete type arguments such as Vector<number> and declare reusable records with type Pair<T>.

05 / FAILURE

Errors are in the type.

Result of T makes recoverable failure visible. Return Ok(value) or Err(message); ? propagates a compatible result.

TYPE CHECKING

Structured constraints validate the program before MIR lowering.

DETERMINISTIC REPAIR

An eligible mismatch can be rebuilt and rechecked before it becomes fatal.
